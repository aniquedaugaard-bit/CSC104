Corrine's Sweet Creations Website

Created by: Anique Daugaard

Course: CSC104
Final Project

Website Description:
This website was created to showcase Corrine's Sweet Creations, a home bakery specializing in custom cakes, cupcakes, macarons, cake pops, cookies, cinnamon rolls, and other homemade desserts. The website provides information about the business, displays examples of baked goods, lists menu items with pricing, and allows customers to submit a custom order request through a contact form.

Files Included:

HTML Files

* index.html
  Home page welcoming visitors to the website and introducing Corrine's Sweet Creations.

* about.html
  Provides information about Corrine, her passion for baking, and the custom desserts she creates.

* gallery.html
  Displays a gallery of cakes, cupcakes, cookies, macarons, and specialty desserts organized into different categories.

* menu.html
  Lists bakery items and starting prices in an easy-to-read table.

* contact.html
  Displays Corrine's contact information and includes a custom order request form for customers.

CSS Files

* v2corrinesstyle.css
  Controls the layout, colors, fonts, navigation, gallery styling, tables, forms, and overall appearance of the website.

Images Folder

* Contains all photographs used throughout the website including cakes, cupcakes, cookies, macarons, cake pops, cinnamon rolls, and other baked goods.

Website Features

* Five linked HTML pages
* Navigation menu on every page
* External CSS stylesheet
* Responsive gallery layout
* Menu pricing table
* Customer contact form
* Specialty script font for headings and captions
* Consistent blush and beige color theme
* Images and descriptive text on every page

This website was developed using HTML5 and CSS and uploaded to GitHub as the final project for CSC104.
